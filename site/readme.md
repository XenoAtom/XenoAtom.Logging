# XenoAtom.Logging

High-performance structured logging for .NET, built for zero-allocation hot paths, predictable async throughput, and rich terminal output.

## Get Started

- [User guide](docs/readme.md)
- [NuGet package](https://www.nuget.org/packages/XenoAtom.Logging/)
- [GitHub repository](https://github.com/XenoAtom/XenoAtom.Logging/)

## Core highlights

- Zero-allocation hot-path logging with interpolated handlers and generated APIs
- Sync-first runtime with optional async queue and overflow policy control
- Structured properties/scopes and source-generated log methods
- File rolling and JSON sinks
- Rich terminal and `LogControl` output via [XenoAtom.Terminal.UI](https://xenoatom.github.io/terminal)
